Filtrations

Filtrations on ReLU polyhedral cell complexes.

Filtration values are attached combinatorially to meta-graph cells wherever possible. Geometric evaluation (interior points, distances to training data) is optional and localized to the built-in filtrations that need it.

class relucent.filtration.AffineOutputFiltration(*, row: int | None = None, combine: Literal['last', 'diff'] = 'last', lower_star: bool = True)

Bases: Filtration

Filtration by an affine output functional w^T x + b on each cell.

Uses the per-region affine map Polyhedron.W, Polyhedron.b (from sign sequences) and one interior point per cell. By default, lower_star extends values to higher cells by max over faces (sublevel-set convention on the face poset). Set lower_star=False to rank each cell only by its representative point.

class relucent.filtration.Filtration

Bases: ABC

Base class for filtrations on cells of a Complex.

Subclasses implement raw_cell_value() for each meta-graph node. Values are then promoted to a lower-star (sublevel-set) filtration on the face poset via lower_star_extension() when lower_star is True (the default).

class relucent.filtration.LogitSublevelFiltration(*, row: int | None = None, binary: bool = False, lower_star: bool = True)

Bases: AffineOutputFiltration

Sublevel-set filtration induced by a scalar logit (last output or class difference).

By default (lower_star=True), raw logit values on 0-cells are extended by max over faces so a cell enters the sublevel set when all its faces are present. Set lower_star=False to rank each cell only by the logit at one interior point.

class relucent.filtration.NeuronActivationFiltration(shi: int, *, target: Literal[-1, 0, 1] = 1, active_value: float = 0.0, inactive_value: float = 1.0)

Bases: Filtration

Filtration by whether a ReLU unit has a prescribed sign on a cell.

Purely combinatorial: reads the cell sign sequence from the meta-graph. Cells matching target receive value 0 and enter the filtration first; non-matching cells receive 1.

class relucent.filtration.TrainingDistanceFiltration(training_points: ndarray, *, ord: int | float = 2, lower_star: bool = True)

Bases: Filtration

Filtration by distance from a cell representative point to training data.

Uses one interior point per cell and numpy norms—no explicit Voronoi or distance-transform geometry. By default, lower_star extends each cell’s value by max over faces (sublevel-set convention).

relucent.filtration.lower_star_extension(meta: Any, raw_values: dict[Any, float]) → dict[Any, float]

Promote vertex values to all cells by f(σ) = max_{τ face of σ} f(τ).

Processes cells in increasing dimension so each value is the maximum of its own raw value and all codimension-one faces already processed.
