relucent.persistence.compute_persistent_homology(cplx: Complex, filtration: Filtration, *, compactify: bool = False, respect_finite: bool = False, lower_star: bool | None = None, verbose: bool = False) → PersistenceDiagram

Compute persistence pairs for a filtration on a Complex.

Cells are ordered by (filtration value, dimension, stable node key). Boundary columns are built from meta-graph face incidences (same convention as relucent.topology.get_betti_numbers()).

Parameters:

  • cplx – Polyhedral complex from breadth-first search / exploration.

  • filtration – Filtration assigning a real value to each cell.

  • compactify – If True, drop face incidences with fewer than two cofaces (Borel–Moore style), matching get_betti_numbers().

  • respect_finite – Restrict to cells with finite is True on the meta-graph.

  • lower_star – If True, extend values to higher cells by f(σ) = max_{τ ≤ σ} f(τ) (sublevel-set convention). If None, use filtration.lower_star (default True for built-in filtrations).

  • verbose – Progress lines on stderr.

Returns:

PersistenceDiagram with finite and essential pairs over GF(2).

relucent.persistence.betti_at_filtration_end(diagram: PersistenceDiagram, *, margin: float = 1.0) → dict[int, int]

Betti numbers after all cells in diagram have entered the filtration.

Evaluates betti_curve() at max(cell_filtration) + margin, which matches the homology of the full meta-graph complex when paired with a constant filtration.

Parameters:

  • diagram – Persistence diagram from compute_persistent_homology().

  • margin – Added to the maximum cell filtration value to form the evaluation threshold (must be positive if all values are equal).

Returns:

{k: β_k} including dimensions with β_k = 0.

relucent.persistence.betti_curve(diagram: PersistenceDiagram, thresholds: ndarray | list[float], *, max_dim: int | None = None) → dict[int, ndarray]

Betti numbers β_k at each filtration threshold (right-continuous counting).

Parameters:

  • diagram – Output of compute_persistent_homology().

  • thresholds – Filtration values at which to evaluate β_k.

  • max_dim – Maximum homological dimension to report (default: max in pairs).

Returns:

{k: array of length len(thresholds)} counting classes with birth ≤ t < death.
